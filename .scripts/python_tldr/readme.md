This script automatically generates a "Too Long; Didn't Read" (TL;DR) summary for Markdown files and injects it into the file. It uses Google's Gemini AI to generate the summaries.

## üöÄ Why This Script Exists

Providing a TL;DR section at the beginning of a long blog post or documentation page can significantly improve user experience. Readers can quickly grasp the main points of the content before deciding to read the entire text. This script automates the process of creating and adding these summaries.

## üõ†Ô∏è Features

*   Recursively finds all `.md` files in a given directory.
*   Uses Google's Gemini API to generate a concise TL;DR summary.
*   Injects the generated TL;DR into the Markdown file.
*   Skips files that already contain a TL;DR summary.
*   Can process a single file, a directory, or the entire blog.
